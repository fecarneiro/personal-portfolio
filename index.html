Sobre

Desenvolvedor de Software com foco na criação de APIs RESTful robustas e eficientes, utilizando principalmente Node.js com Express.js e Java com Spring Boot. Minha experiência em front-end inclui a criação de sites e interfaces de aplicação com HTML, CSS e JavaScript.

Minha trajetória anterior como Scrum Master me proporcionou uma base sólida em metodologias ágeis, organização de equipes e processos de desenvolvimento de software. Essa vivência me permite ter uma visão estratégica sobre o ciclo de vida do produto, desde a definição do roadmap e a experiência do usuário (UX) até o relacionamento com stakeholders, utilizando ferramentas como o Jira.

Atualmente, aplico meus conhecimentos técnicos em projetos práticos, seguindo os princípios do Clean Code, utilizando versionamento com Git e automação de ambientes com Docker. Tenho experiência com bancos de dados relacionais como PostgreSQL e SQLite, e na implementação de autenticação com JWT e criptografia com Bcrypt.
